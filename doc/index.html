Como installar Ralf.php

Para poder hacer uso de y de los componentes ralf.php es requerido primero hacerse de una copia del ralf.php para lo que puedes descargar desde aquí (https://github.com/alfa30/ralf.php#descargar).

Dentro del archivo comprimido existe una carpeta llamada “dist”, la cual hay que ubicar en una ruta desde la cual puedas usarla. Por ejemplo:

/ruta/

Implementar Ralf.php

Para usar ralf.php en nuestro proyecto es necesario incluir en todos nuestros proyectos relacionados el archivo ralf.php que se encuentra dentro de nuestra carpeta dist/ que se descargó previamente.

Para ello utilizamos la sentencia include de php. Por ejemplo: 

Include '…/dist/ralf.php';

Componentes de ralf.php

Los componentes nos ayudaran al momento de desarrollar nuestras aplicaciones en php. Integrando características y herramientas que hacen más claro el código y hacen más sencillo el desarrollo.

Todos los componentes en ralf.php por defecto no se pueden utilizar tan solo instanciando a ralf.php si no que se debe hacer uso del método estático que se implementa en ralf.php.

ralf::import("dom.model.themeHtml");

Esta instrucción importara el modulo que corresponde al componente que deseemos utilizar.

dom.html PHP Simple HTML DOM Parser

Autor
S.C. Chen <me578022@gmail.com>
Versión
v1.5
Link
http://simplehtmldom.sourceforge.net/

Un simple analizador DOM HTML PHP escrito en PHP5 +, compatible con HTML válido, y proporciona una manera muy fácil de manejar elementos HTML.

Fuente

Este módulo incorpora una uno de los componentes más robustos dentro de ralf.php el cual permite leer el DOM del código HTML.
